Toto je největší nadpis napsaný tagem <h1>

Toto je druhý nadpis napsaný tagem <h2>

I toto je nadpis napsaný tagem <h3>

Ještě menší než ten předchozí a použitý tag <h4>

A ještě ještě menší psaný tagem <h5>
Tohle je ten opravdu nejmenší nadpis, který se dá zapsat tagem <h6>

Tagy <h1-h6> slouží ke psaní nadpisů. Značí se písmenem "h", protože vychází z anglického slova Heaading.
Tag <p> slouží ke psaní odstavců. Značí se písme>nem "p", protože vychází z anglického slova Paragraph.

Další tagy, které se používají v Syntaxu HMTL jsou (seřazené odshora):

<!DOCTYPE html> - užívá se k tomu, aby prohlížeč vykresloval stránku ve standardním režimu
<html> ... </html> - označuje začátek a konec html dokumentu
<head> ... </head> - hlavička dokumentu, do které se dávají informace o názvu karty v prohlížeči
<title> ... </title> -název karty v prohlížeči. Musí být umístěný v tagu <head>
<body> ... </body> - obsahuje veškerý zobrazovaný obsah stránky
